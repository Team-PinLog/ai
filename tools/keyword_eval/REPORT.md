# Keyword 매칭 평가 리포트

프리셋 `data/keyword_preset.yaml`(27개)에 대한 매칭 테스트 결과. 임베딩 `text-embedding-3-small`(GMS), 판정 `gpt-5-mini`(GMS).

> ⚠️ 샘플 `samples.yaml` 35개는 내가 프리셋을 알고 쓴 **임시 데이터**(self-reference 편향). 수치는 경향으로만 해석하고, 팀이 프리셋 안 보고 쓴 샘플로 교체하면 유효한 측정이 된다.

## 테스트 A — 프리셋 자기 중복

- **병합 후보(cosine ≥ 0.9): 0건.** 27개 프리셋 독립성 OK, 병합 불필요.
- 최근접 이웃 상위: `WITH_PARTNER↔DATE_COURSE` 0.578, `QUIET↔COZY` 0.541, `ANNIVERSARY↔CELEBRATION` 0.515 — 의미상 인접하나 별개.
- 프리셋↔프리셋 분포: p50 0.315 / p90 0.422 / max 0.578.

## 테스트 B — 커버리지 (K=10, floor 0.30)

- 미매칭율 2.9%(1건, 의도적 무매칭 샘플) / 쏠림도 max-share 10%·Gini 0.286 / 사각지대 0 / Recall@10 30/30(편향·참고용).
- top-1 분포: p10 0.357 / p50 0.547 / p90 0.658 → 진짜 매칭 대체로 0.45+, 부대시설·무관 입력은 0.30 부근에서 갈림.
- `QUICK_STOP`이 약한 후보로 자주 등장(23회) → 판정 단계 거름 여부가 관건이었고, C-1에서 정상 기각 확인.

## 테스트 C-1 — 프롬프트 안정화 (gpt-5-mini)

35샘플, K=10. **최종 설정: floor 0.30 + 부대시설/서비스 제외 규칙.**

- 스키마 위반 0 / 파싱 실패 0 / 과잉 선택(>3) 0. 구조화 출력 안정.
- 선택 개수 mean 1.32(분포 0:4 / 1:18 / 2:9 / 3:3).
- **판정 계층이 임베딩 후보 노이즈를 교정함**: "여자친구랑…"에서 top-1 후보가 `WITH_FAMILY`(0.485)였으나 판정이 기각하고 `WITH_PARTNER` 선택.

### 발견과 교정

| 발견 | 원인 | 교정 |
|---|---|---|
| "주차가 넓어서" → SPACIOUS 오탐 | 부대시설 언급의 단어 수준 오버제너럴 | **프롬프트에 부대시설/서비스 제외 규칙 추가** → 판정이 기각 |
| floor 0.35로 올리면 "시험기간에 살다시피"(STUDY_WORK) 유실 | 간접 표현이라 임베딩 약함(0.30~0.35) | **floor 0.30 유지** — 프롬프트로 정밀도 보완(하한 상향 대신) |
| "여자친구랑 기념일에…"에서 ANNIVERSARY 누락 | ANNIVERSARY 임베딩 0.265(#12), 문장 전체가 date/meal에 지배 → 후보 밖 | **알려진 한계로 수용**. 하한을 0.26까지 내리면 노이즈 급증. ANNIVERSARY는 PRIVATE_ONLY라 누락의 해가 작음 |

### 판정 모델 관찰 (gpt-5-mini)

- confidence가 과신 경향(mean 0.94, std 0.03~0.06) → **변별력 낮음**. MVP에서 confidence를 강한 랭킹 신호로 쓰지 않는 게 안전.
- 지연 mean ~4s/call(추론 모델). 비동기 처리라 파이프라인엔 무해하나, C-2에서 경량 비추론 모델과 지연·크레딧 비교 가치 있음.

## 결론

- **프리셋 보정 불필요**(A/B 건전, 병합·삭제 대상 없음).
- **프롬프트 확정**: `prompts/keyword_judgment.md`(부대시설 규칙 포함).
- **후보 검색 파라미터**: `KEYWORD_CANDIDATE_TOP_K=10`, **하한 0.30**.
- **판정 모델**: gpt-5-mini로 안정 동작 확인. 최종 확정은 **C-2 모델 비교**(gpt-5-mini / gpt-5-nano / gemini-flash 등, 5개 지표)에서.

## 남은 것 (C-2)

- Gemini는 네이티브 `generateContent` 포맷이라 별도 어댑터 필요(현재 openai provider는 GPT 계열만).
- GMS 실제 모델 id 확인 후 `test_c_judge.py`의 `MODELS` 목록 확정.
- 팀 실제 샘플로 B/C 재측정 시 Recall이 유효해짐.
