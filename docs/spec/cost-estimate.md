# 운영 비용 추정 (토큰 → 비용)

근거 데이터: 테스트 C-2 토큰 측정(`tools/keyword_eval/REPORT.md`, 35샘플, K=10, floor 0.30).

> ⚠️ **단가는 공개가 근사치입니다.** 실제 청구는 GMS 게이트웨이의 크레딧 단가로
> 이루어지며, 모델별 크레딧 환산율표는 아직 확보하지 못했습니다. 환산율표를
> 받으면 아래 §4 공식에 대입해 재계산합니다. 현재 수치는 규모 감을 잡기 위한
> 근사이며 청구 금액이 아닙니다.

## 1. Context 1건 처리의 호출 구성

`/context/process`가 Context 1건을 처리할 때 발생하는 외부 모델 호출:

| 단계 | 호출 | 비고 |
|---|---|---|
| Embedding | 본문 1회 | `text-embedding-3-small`. 재사용(partial-resume) 시 0회 |
| 판정 | LLM 1회 | 후보 0개면 호출하지 않음(0건 처리) |

검색(`/search`)은 질의당 Embedding 1회이며 LLM 호출이 없습니다.

## 2. 측정된 토큰 프로파일 (호출당)

판정은 C-2에서 실측했습니다(34회 호출 평균). Embedding 본문 토큰은 입력 길이에
따라 변하므로 가정치를 병기합니다.

| 항목 | 입력 tok | 출력 tok | 근거 |
|---|---|---|---|
| Embedding(본문) | 30 ~ 100 | — | 짧은 한국어 문장 가정(측정 대상 아님) |
| 판정 `gemini-2.5-flash` | 717 | 27 | C-2 실측(24398/34, 916/34) |

판정 입력 717 tok에는 SYSTEM 프롬프트 + 후보 10개 메타(display_name·description·
examples)가 포함됩니다. 출력 27 tok은 `selected` 배열(평균 1.29개)입니다.

## 3. 확정 모델(gemini-2.5-flash) 운영 비용 근사

공개가 근사(2025 기준, USD / 1M tokens):

| 모델 | input | output |
|---|---|---|
| text-embedding-3-small | 0.02 | — |
| gemini-2.5-flash (thinking off) | 0.30 | 2.50 |

Context 1건(본문 50 tok 가정, 후보 있어 판정 1회):

```
Embedding : 50  × 0.02/1e6                     = $0.0000010
판정      : 717 × 0.30/1e6 + 27 × 2.50/1e6      = $0.0002826
────────────────────────────────────────────────────────────
합계                                            ≈ $0.000284 / Context
```

규모별 (판정률 100% 가정, 상한):

| Context 수 | 근사 비용(USD) |
|---|---|
| 1,000 | $0.28 |
| 10,000 | $2.84 |
| 100,000 | $28.4 |

본문 길이 민감도(임베딩 tok)는 전체 비용에서 미미합니다 — 판정이 비용의 99%를
차지하므로, 본문이 30이든 100이든 Context당 비용은 $0.00028 부근입니다.
후보 0건(판정 스킵) 비율만큼 실제 비용은 위 상한보다 내려갑니다.

## 4. GMS 크레딧 환산 공식 (단가표 확보 시 대입)

```
Context당 비용 = Σ(단계별 tok × 해당 모델·방향 크레딧단가)
             = emb_in × rate(embedding)
             + judge_in × rate(judge, input)
             + judge_out × rate(judge, output)
```

`rate(...)`에 GMS 크레딧 단가를 넣으면 청구 기준 비용이 됩니다. tok 프로파일
(§2)은 모델 고정이므로 단가만 갈아끼우면 됩니다.

## 5. 모델 간 상대 비교 (토큰 총량 기준)

C-2 총 토큰(35샘플 34호출)으로 본 상대 순위입니다. gemini가 최소이며, 이것이
확정 근거의 일부입니다(REPORT §C-2).

비용은 §4에 단가를 대입해 확정합니다.

## 6. 남은 것
